# What'sUp — Public Release Binaries

This repository hosts the signed & notarised `.dmg` builds of
**[What'sUp](https://findwhatsup.com)** — the macOS menu-bar tool that keeps
track of the dev servers your AI assistant keeps spinning up in the
background.

## Downloads

Binary downloads live under **[Releases](../../releases)**. The latest DMG is
always reachable at:

```
https://github.com/HMyaoyuan/whatsup-releases/releases/latest/download/WhatsUp.dmg
```

This URL is what `findwhatsup.com` points to, so uploading a new release tag
automatically updates the site's download button.

## Source

The **source code** lives in a separate, private repository. This repo only
carries release artifacts, release notes, and signed appcast metadata for
in-app updates — it's intentionally empty of code.
